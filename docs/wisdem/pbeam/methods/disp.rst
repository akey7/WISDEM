_pBEAM.Beam.displacement
========================

.. method:: Beam.displacement()

    Computes displacement of structure at each node

    Returns
    -------
    x : ndarray (m)
        displacement in x-direction at each node
    y : ndarray (m)
        displacement in y-direction at each node
    z : ndarray (m)
        displacement in z-direction at each node
    theta_x : ndarray (rad)
        rotation in theta_x direction at each node
    theta_y : ndarray (rad)
        rotation in theta_y direction at each node
    theta_z : ndarray (rad)
        rotation in theta_z direction at each node
